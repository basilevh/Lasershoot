# Lasershoot

This is the repository for the DLW Lasershoot Hackathon.

## tests
dependencies: mysqlclient:
On debian based systems:
`sudo apt-get install libmysqlclient-dev`
on macos:
`brew install mysqlclient`
run with `python3 setup.py test`
